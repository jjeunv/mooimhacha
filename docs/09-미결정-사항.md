# 09. 미결정 사항

구현 착수 전 또는 PoC 후 결정이 필요한 항목.

## STT 관련 (Web Speech API — PoC 후 미세조정)

- [ ] `confidence` 보정식 — 기본은 `SpeechRecognitionAlternative.confidence`(0~1) 그대로 저장 결정. 브라우저별 0/미제공 케이스 보정·대체식만 PoC 후 확정
- [ ] 신뢰도 등급(high/medium/low) 임계값 정밀화 — 임시 기준은 확정(아래 이력), 수치는 운영 중 조정
- [ ] ~60초 재시작 시 발생하는 공백 구간 손실의 정밀 기준 (기본 onend 자동 재시작+겹침 결정)

## 도메인·HTTPS (웹 전환에 따른 신규)

- [ ] 도메인 발급·DNS 운영 주체 ([10](10-AWS-인프라-제약.md))
- [ ] 프론트 정적 서빙 방식 — EC2 단일 서빙(기본) vs 분리

## V2 (데스크탑 전환 후)

- [ ] **faster-whisper 모델 크기 선택** (`tiny` / `base` / `small`) — RealtimeSTT 도입 시 결정
- [ ] VAD 임계값 (RealtimeSTT 내장 VAD 파라미터)

### STT (2026-05-29, Web Speech 전환분)

- 지원 브라우저 → **Chrome/Edge(Chromium) 전용**, Firefox/Safari는 안내 페이지
- 발화 전송 단위 → **isFinal(확정 발화)만 전송**, interim 미표시
- 마이크 선택 → **OS 기본 마이크 따름**(Web Speech 장치 지정 불가) + 음량 테스트만 제공
- `confidence` → **Web Speech `alternative.confidence`(0~1) 그대로 저장**, null 허용
- STT 실패 판정 → **`onerror`(network/no-speech/aborted) + onend 재시작 실패**를 `anomaly_events.stt_failure`로 기록, 활성 윈도우서 차감
- 신뢰도 등급(임시) → 오디오손실<5% & STT실패<10% = high, ≤20% = medium, 그 외 low (운영 중 조정)

### 기여도 산정 (2026-05-29, 2트랙 재설계) — 상세 [06](06-기여도-산정.md)

- 기여도 구조 → **2트랙 + 최종 종합**: 회의 기여도(발언×0.6 + 참석×0.4) / 테스크 기여도(완료율·마감준수) 분리, 최종 = 테스크×w + 회의 종합×(1−w)
- 가중치 → 액션 난이도는 전역 고정. **발언:참석(기본 0.6/0.4)·테스크 비중(`final_task_weight`)은 팀 설정 조정 가능** (2026-06-20 변경: 기존 '발언:참석 전역 고정·테스크 비중 생성 시 고정'에서 팀별 조정 허용으로 전환. `weight_speech_in_meeting`/`weight_attend_in_meeting` 추가)
- 팀장 보정 → **최종 종합에만 ×배율**(`leader_bonus_multiplier`, 기본 1.0, 1.0 캡)
- 발언 비중 → **1/N 정규화**(참석자 균등 분량이면 만점, 무발언 참석자도 N 포함)
- 액션 난이도 → **상/중/하 = 3/2/1** 가중 평균 (회의 중 기본 '중', 회의 후 보정)
- 마감점수 → **일자별 공식 3종**: standard `max(0,1−0.2×초과일수)` / lenient `max(0,1−0.1×초과일수)` / strict(초과 즉시 0)
- 기한 전 미완료 액션 → **평가 보류**(완료율·마감준수 분모서 제외), 기한 지나면 0
- 테스크 스코프 → **`action_items.team_id`**(회의와 분리, 조인 없이 user×team 집계), 스냅샷 테이블 미사용(라이브 계산)
- 누적 점수 가중 → **회의 시간 가중 평균** Σ(점수×회의시간)/Σ(회의시간)
- 최소 회의 길이 → **5분 미만 누적 미반영** (`team_settings.min_meeting_minutes` 기본 5)
- 3축→2트랙 회의 유형 → **`regular`/`partial`** (정규만 산정, 부분은 누적 미반영. 테스트 모드 별도 미제공)
- 누적 결석 제외 → **사유결석(팀장 승인)만 분모 제외, 무단결석은 0점 반영**

### 첨부 (2026-05-29)

- 테스크 첨부 → **외부 링크만**(`action_items.link_url`). S3·`attachments` 테이블·직접 업로드는 미사용(정식 드라이브 연동은 V2)
